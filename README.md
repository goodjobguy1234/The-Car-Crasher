# Car Clasher

## About Project
this project is created as a first term project for game development subject.
## Player controll:
    A - turn left
    D - turn right
    spacebar - shoot bullet
    v - switch camera view

## GamePlay
there are 3 type of obstacle:
- van
- rock
- crate
 
player cannot crash with vans and rock but only crates.
